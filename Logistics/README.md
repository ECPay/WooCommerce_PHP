# WooCommerce  綠界物流
---

## 1. 介紹

  - 綠界科技超商取貨付款外掛套件，提供合作特店以及個人會員使用開放原始碼商店系統時，無須自行處理複雜的檢核，直接透過安裝設定外掛套件，便可快速介接綠界科技物流整合系統，使用方便快速的商品運送機制。
    
     - 若有同時使用金流模組時,更新模組的話必須連金流（含開立電子發票版本）、物流一起更新才能使用


## 2. 適用版本
  
- Wordpress Version 4.5.2
- Wordpress Version 4.4.3
- Wordpress Version 4.3.4
- WooCommerce Version 2.5.5
- WooCommerce Version 2.4.13
